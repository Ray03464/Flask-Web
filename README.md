# Flask-Web

A simple full-stack web application built with Flask as part of my web development learning journey.

## Features
- User authentication (login and registration)
- CRUD operations for managing data
- RESTful API endpoints
- File upload functionality

## Technologies Used
- Python
- Flask
- MySQL
- SQLAlchemy
- HTML
- CSS
- Jinja2

## Purpose
This project was created to practice building full-stack web applications using Flask, including backend development, database integration, and basic authentication.

## Status
This project is still being improved as I continue learning web development.
